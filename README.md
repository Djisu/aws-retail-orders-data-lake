# AWS Retail Orders Data Lake

A beginner-friendly data engineering project that simulates building a simple retail orders data lake pipeline using Python, PySpark, and Apache NiFi concepts.

## Project Overview

This project demonstrates a basic end-to-end data engineering workflow for ingesting, processing, and organizing retail order data in a data lake style structure.

The goal is to practice foundational data engineering skills such as:

- Data ingestion
- Data transformation
- File-based data lake organization
- Workflow orchestration concepts
- Version control with Git and GitHub

## Tools & Technologies

- Python 3
- PySpark
- Apache NiFi
- Git & GitHub
- macOS Terminal

## Project Objectives

- Load retail order data from source files
- Process and transform the data using PySpark
- Organize outputs into a simple data lake structure
- Practice using Apache NiFi for ingestion/workflow concepts
- Document the project in a professional GitHub repository
